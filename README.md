# Best of Cebu - Project Overview

## Vision
A premium, magazine-style digital platform showcasing the best of Cebu, Philippines.

## Technical Stack
- **Framework**: Next.js / React
- **Styling**: Tailwind CSS
- **Content**: MDX / Contentlayer (planned)
- **Deployment**: Vercel (planned)

## Phases
1. **Phase 1: Foundation** (Repo setup, base styles, layout) - **IN PROGRESS**
2. **Phase 2: Editorial Conversion** (Converting raw content to magazine layout)
3. **Phase 3: Features** (Interactive maps, curated lists, business directory)
4. **Phase 4: Optimization** (SEO, Image optimization)
